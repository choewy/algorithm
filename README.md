# Algorithm

## Typescript

```bash
# initialize
npm ci

# run main.ts
npm run ts

# run test/**/*.spec.ts
npm run ts:test
```

## Python(3.10, 3.11)

```bash
# create venv
python3 -m venv .venv

# activate venv
source .venv/bin/activate

# install packages
pip3 install -r requirements.txt

# run main.py
npm run py

# deactivate venv
deactivate
```
